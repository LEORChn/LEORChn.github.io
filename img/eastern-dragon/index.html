Startup parameters:
	?feature=controls	Display this controls
	?feature=autoplay	Apply "Keep growing"
	?feature=controls,autoplay	Apply multi parameter
	
	You can find the keys on the keyboard:
	Home		Go to top of the page
	End			Go to bottom of the page
	Page Up		Scrolling up the page
	Page Down	Scrolling down the page

	Thanks pulexart.com for sharing.



	启动参数：
	?feature=controls	显示控制面板
	?feature=autoplay	应用“持续生长”设置
	?feature=controls,autoplay	连续应用指令的示例
	
	你可以在键盘上找到如下按键：
	Home		回到顶部
	End			去往底部
	Page Up		向上翻页
	Page Down	向下翻页

	感谢作者 pulexart.com 的分享
